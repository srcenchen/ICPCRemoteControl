# ICPC 无锡学院 Linux 集控系统
## 技术选择
我们后端选择 go + sqlite3(纯go版本)  
前端选择 jquery 嵌入到 server 端
## 文件目录构建
包含: cmd/client.go|server.go  
internal/biz|data|model|server|service
go的标准化开发  
## 功能需求与流程
1. 首先我们启动服务端，服务端部署在Linux 你要通过 `avahi-publish` -a -R icpc-server.local 0.0.0.0 发布地址
2. 客户端与服务端的通信使用 icpc-server.local 写死。但同时有冗余操作。
3. 冗余操作：客户端启动时，在用户目录创建一个 server 空文件，如果这个文件不为空，就使用这个文件文本（是一个ip）作为服务端
4. 建立可靠的长连接与服务端，有断线重连机制，服务端要记录选手机的信息。
5. 对于选手机的信息，你可以通过：fastfetch --format json > /tmp/spec.json 获取

## 对于服务端前端的要求
使用JQuery 嵌入到服务端中。要有以下页面：  
首页（包括目前设备状态）  
设备管理（看到设备的简略情况），允许点进去查看详情。  
命令执行，可以去执行命令，要求：输入命令的页面要有高亮。
